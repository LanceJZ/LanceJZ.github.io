Welcome to Panther Games

Updated on June 6, 2015

 

This is Lance Zimmerman and I created this web site to share my latest shared video game projects.

I will post all of my video game projects on Git-hub, all open source. I hope to share what I've learned so others may learn from it. Remember, sharing is caring. So far I have two game projects that are active. Vector Asteroids and Vector Omega Race.

I'm currently working on rebooting a game I did using XNA 2 years ago named Base Defender. I'm making it in C++ using SFML2. I will upload it as soon as it is at a point that it does something. I am behind because my desktop HDD failed, and I've been saving up money to replace it. Don't worry I had the project backed up.

I started learning C++ in 2012. In that time I have learned much, but I also know I have so much more to learn. It helped so much that I already had a good grasp of programming. I started learning to program video games using C# using XNA back in 2009.
